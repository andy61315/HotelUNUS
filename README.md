# HotelUNUS
a website for hotel management system, designed by a seven-people group during Institute for Information Industry(資策會).

## Main Function
* Booking System
* Room Service (for meal order)
* Reservation for restaurant
* Meal order in restaurant
* Diary management

## Languages / Techniques
### front-end
* JavaScript 
* jQuery
* Ajax
* JSON

### back-end
* Java
* MVC model

### DataBase
* Oracle
* Redis

### Environment / Server
* Windows 10
* Tomcat

### Development Tools
* Eclipse
* Sqldeveloper
* Sublime
* RedisDesktopManager
